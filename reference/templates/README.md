# Daily Report Templates

本目录只放日报 Markdown 版式模板，不放判断规则、不放某次运行的具体变量值。

使用约定：

- workflow 文件负责说明何时读取模板、哪些章节可条件省略。
- 模板文件负责稳定版式、章节顺序、固定署名和占位符。
- 所有路径相对 daily-report skill 根目录。

## 文件清单

- `main-body.md`：第 2.0 步正文主体模板。
- `token-stats.md`：Token 统计章节模板。
- `deliberation-appendix.md`：审议过程折叠附录模板。
- `signature.md`：日报末尾署名模板。
- `empty-report-minimal.md`：空日最简形态参考。
